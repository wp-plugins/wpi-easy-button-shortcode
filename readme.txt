 Create buttons anywhere in wordpress using button shortcode [wpi_easy_button] 

== Description ==

1. Create buttons anywhere in wordpress using button shortcode [wpi_easy_button]
2. For String attributes use Double or Single Qoutes like text="Sample Products"

* Correct Shortcode examples: 
` [wpi_easy_button link='/products/' text='products']`

* Shortcode: 
` [wpi_easy_button]`

== Installation ==

1. Upload folder `wpi-easy-button-shortcode` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Shortcode [wpi_easy_button]

* Correct Shortcode examples: 
` [wpi_easy_button link='/products/' text='products']`
